Bæredygtig UX

Der er mange måder at styrke bæredygtigheden på i forbindelse med, brugeroplevelsen (UX). Disse tiltag øger ikke kun effektiviteten på den enkelte website, men reducerer også miljøpåvirkningen.

Reducer unødtvendig pageload

Ved at reducere størrelsen på billeder og andre medier kan vi reducere indlæsningstiden på vores hjemmeside betydeligt. Dette vil forbedre brugeroplevelsen, især for brugere med langsom internetforbindelse eller mobile enheder.

Fjern forældelet content

Slet gammelt indhold, der kan forvirre brugerne. Brugerne skal kun eksponeres for det mest nødvendige og aktuelle content. Derved bliver pageload også højere.

Minimere kliks

Reducer antallet af klik i brugerrejsen: Vi kan redesigne navigationsstrukturen på vores hjemmeside for at sikre, at brugerne kan nå deres mål med færre trin og klik. Dette kan omfatte at reducere antallet af undermenuer eller at tilføje tydelige call-to-action-knapper på vigtige sider, ergo forenkle navigationen.




Signalere gennemsigtighed

Signalere gennemsigtighed fra virksomhed til kunde: som virksomhed kan vi implementere sektioner på vores hjemmeside, der tydeligt viser vores bæredygtige initiativer. Dette kan omfatte oplysninger om vores CO2-aftryk, brug af genanvendelige materialer eller vores deltagelse i grønne programmer og partnerskaber.

Videreformidling

Del bæredygtig tiltag med andre og gør opmærksom på CO2-aftrykket af de digigtale produkter. Jo flere der er med til at gøre en forskel jo bedre.

Økologiske konsekvenser

Informere om de økologiske konsekvenser: Vi kan tilføje sektioner, der informerer brugerne om de økologiske konsekvenser af deres handlinger. Dette kan omfatte oplysninger om skovrydning, forurening eller klimaforandringer og opfordringer til handling eller deling af ressourcer til yderligere læsning.
